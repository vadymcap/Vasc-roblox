# VASC — Roblox Plugin

VASC for Roblox Studio is the in-engine counterpart of the VASC toolchain. It enables two-way sync of code and other instances with their properties, and provides additional features such as place porting, code execution, and external editor integration.

It is part of the VASC project, a fork of [Vasc](https://github.com/vasc-rbx/vasc), extended with additional features and improvements.

> The plugin requires either the [VASC CLI](https://github.com/vadymcap/vasc) or the [VASC VS Code](https://github.com/vadymcap/vasc-vscode) extension to function.

## Features

- **Two-way sync** — keep code and instance properties in sync between Roblox Studio and your external editor in real time
- **Place porting** — port places between different projects with ease
- **Code execution** — run scripts directly from your external editor inside Studio
- **External editor integration** — works seamlessly with VS Code and any editor using the VASC CLI

## Related Packages

| Package | Description |
|---|---|
| [**vasc**](https://github.com/vadymcap/vasc) | Core CLI — handles all processing and syncing logic |
| [**vasc-vscode**](https://github.com/vadymcap/vasc-vscode) | VS Code extension — GUI wrapper around the CLI |
| **vasc-roblox** *(this repo)* | Studio plugin — required for live sync to function |
